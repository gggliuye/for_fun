**1. 辩证法**

* 辩证，对立思绪的碰撞。
* 能动主义 ，人类官能性。
* 变化，万物是变化运动的。

“矛盾”不是逻辑上都矛盾，事物内部相互冲突的基本过程这一性质。矛盾导致了变化。
从相互联系，相互作用的背景下理解客体的存在。
“反者道之动”。
辩证的任务是识别具体社会过程中的特殊矛盾倾向（如果有的话）。

警示：

* 社会过程内存在的冲突本身并不足以将这些冲突确认为矛盾。
* 辩证方法自身还不能决定具有矛盾倾向的事件的实际顺序。


在下列情况下，两个概念是辨证相关的：对一个概念的详尽阐述会引起对另一个对立概念的关注，前者隐含着对后者的否定或排斥；
当人们发现对立概念是前者有效性或适用性所必需的；当人们发现真正的理论问题是这两个概念之间的相互关系问题，而真正说明的问题是根据具体情况来决定二者问的相互关系。
